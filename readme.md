 
# Swimbots to prosty kod w python nasladujący ewolucję tworków podobnych do plemników, które w kolejnych generacjach coraz sprawniej poruszaja się.

# Swimbots is a simple Python code that mimics the evolution of sperm-like creatures that move more and more efficiently in subsequent generations.

### W styczniu 2026 zafascynowałem sie pracami Jeffrey Ventrella <https://www.ventrella.com/> i spróbowałem stworzyć własnego swimbota - kilka "plemników" w procesie ewolucji płynie do celu coraz dalej. W każdej kolejnej generacji wybieramy tylko najlepiej płynące swimboty i drogą ewolucji staramy się wyhodować najlepiej pływające.
### Na razie jest to projekt na wczesnym etapie , pojawiaja sie błędy itp

### In January 2026, I became fascinated by the work of Jeffrey Ventrella <https://www.ventrella.com/> and attempted to create my own swimbot – several "sperm" swim further and further towards their destination through evolution. In each subsequent generation, we select only the best-swimming swimbots and, through evolution, strive to breed the best swimmers.
### For now, this is an early-stage project; bugs and other issues arise.
